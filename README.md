# Sendlane

Sendlane is a unified email, SMS, reviews, and forms platform designed for e-commerce brands. It provides a REST API for managing contacts, lists, campaigns, automations, custom fields, sender profiles, tags, and custom integration events, with a focus on revenue-driven email analytics and customer retention.

## API

- **Base URL:** `https://api.sendlane.com/v2`
- **Authentication:** Bearer token (`Authorization: Bearer <api_token>`)
- **Documentation:** https://sendlane.stoplight.io/docs/api-documentation/c53add3c8b16f-overview
- **Postman Workspace:** https://www.postman.com/sendlane/sendlane-s-public-workspace/documentation/xl7jbbs/sendlane-api
